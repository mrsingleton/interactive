This is an example of a filterable table, which a few lines of jQuery makes interactive. It's my version of W3School's jQuery Filters example, which I've customized for my own students and course.

Notes

The filter doesn't target the entire table, it only applies to the tbody tag with the table-content ID, to avoid filtering the table header.

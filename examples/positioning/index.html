Hotspot Positioning

Place dots over an image to create hotspot effect

This page has a few moving parts:

  1. The image div is relatively positioned in relation to the whole web page.
  2. The hotspots are absolutely positioned in relation to the image div.
  3. Dots A and B each have their own unique coordinates.
  4. Width dimensions are set by percentages, not pixels, for device scaling.
